# OKKY Project
[개발자 지식 공유 플랫폼 Okky 사이트](https://okky.kr/) 클론코딩

### 🖥️ 프로젝트 소개
Okky 사이트를 참고하여 만든 개발자 커뮤니티 사이트 입니다.  

### 🧑‍🤝‍🧑 멤버 구성
- Front-end : 1명
- Back-end : 2명

### 🔧 개발 환경
* **Server** : Ubuntu 22.04.3 LTS
* **Web Server** : Nginx 1.18.0
* **DB Server** : GCP(e2-micro 인스턴스 - MySQL)
* **Framework** : Spring Boot 3.2.1 (JDK 17)
* **Build Tool** : Gradle
* **Language** : Java
* **DB** : MySQL + JPA
* **CI/CD** : Jenkins
* **형상관리** : Git
